# NetworkGPT

**NetworkGPT** is an AI-powered Network Engineer built using **Python**, **Streamlit**, and **Google Gemini**.

The project is designed to help network engineers analyze, troubleshoot, secure, and understand network configurations using Large Language Models (LLMs) combined with rule-based analysis and Retrieval-Augmented Generation (RAG).

## Project Goals

* Analyze Cisco network configurations
* Explain configuration commands in plain English
* Perform security audits and compliance checks
* Generate configuration recommendations
* Compare multiple configuration files
* Answer questions about uploaded network configurations
* Generate professional network analysis reports

## Technology Stack

* Python
* Streamlit
* Google Gemini API
* FAISS (RAG)
* LangChain
* Pandas
* Plotly
* Loguru
